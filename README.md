# Система управления пользователями

Проект представляет собой консольное приложение для управления пользователями с использованием базы данных SQLite. Реализованы функции регистрации, аутентификации и просмотра списка пользователей.

## 📋 Функциональность

- Создание базы данных и таблицы пользователей
- Добавление новых пользователей (логин, email, пароль)
- Аутентификация пользователей по логину и паролю
- Просмотр списка всех зарегистрированных пользователей
- Проверка на дублирование логинов

## 🛠 Технологии

- Python 3.14+
- SQLite3
- pytest (тестирование)
- pytest-cov (покрытие кода)
